Prerequisite: Installation of Dev-C++

Execution 1:

1. Refer the PNG images in folder, hanoi, there are four steps.
2. the first step is to open the Dev-C++ project file by clicking. (0_ClickProject)
3. After that, Drag your mouse and click Execute so you could see Execute option. Click it. (1_CompileMain)
4. Subsequently, you should drag your mouse at the same position and click run. (2_RunMain)
5. Finally, the terminal would be opened and type the number of disks, such as n = 3. you could see the result. (3_Result)

Execution 2:

1. click the hanoi.exe.
2. After the terminal pops up, type any number, like n=3.
3. You alse see it.
